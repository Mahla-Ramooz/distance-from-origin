# Distance From Origin

This project calculates the Euclidean distance between a point and the origin (0, 0) using Python.

## Features

- Uses a tuple to store coordinates
- Calculates distance with the Euclidean distance formula
- Displays the point and the calculated distance

## Example Output

text
Point: (4, 3)
Distance from origin: 5.0


## Requirements

- Python 3.x
